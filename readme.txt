*********************************************************************************************************************************************************		  
1-Acceder a la siguiente URL del repositorio en github: https://github.com/alexander-aguilar-b/ecos-psp-tareas-tarea3
2-Descargar el ZIP del proyecto (ecos-psp-tareas-tarea3-master.zip)
3-Descomprimir el archivo anterior.
4-Abrir una ventana de comandos de windows dentro de la carpeta descomprimida en el paso anterior (por ejemplo en C:\Users\edgaguil\Downloads\ecos-psp-tareas-tarea3-master)
5-Ejecutar el comando: mvn package
6-Ejecutar el comando: java -cp target/PSPProgram03-1.0-SNAPSHOT.jar uniandes.ecos.psp.PSPProgram3
7-Ingresar la ruta del archivo de texto plano que conteniene el listado de parejas (x;y) de numeros reales separados por punto y coma (;) entre datos de una misma pareja
y por salto de linea entre parejas de datos.
